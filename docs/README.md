## head

Gets the first element of array.

### Parameters

*   `collection` **[Array][9]** 

### Examples

```javascript
head([1, 2, 3]);
// 1
```

Returns **any** 

## noop

This function do nothing.

### Examples

```javascript
noop();
// undefined
```

## uniq

Creates a duplicate-free version of an array.

### Parameters

*   `collection` **[Array][9]** 

### Examples

```javascript
uniq([2, 1, 2]);
// [2, 1]
```

Returns **[Array][9]**
